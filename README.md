<h2>Building</h2>
<h3>Ubuntu/Debian:</h3>
    <pre>sudo apt-get install build-essential mtools qemu<br/>./compile.sh<br/>cd kernel && make run</pre>
<h3>Arch-Linux:</h3>
    <pre>sudo pacman -S base-devel mtools qemu<br/>./compile.sh<br/>cd kernel && make run</pre>
